EpisodeItem

Main episode list item. Fixed height 72dp row with image thumbnail (largeIncreased corners, 16dp corner radius), content column (title 2 lines, subtitle with date and feed/duration), and action column with play and like toggle buttons. The play button uses EpisodiveIconProgressButton (32dp) and the like button uses EpisodiveIconToggleButton (32dp).

PlayedEpisodeItem

Episode row with playback progress. Contains a 68dp image, title (2 lines), a LinearProgressIndicator with percentage label, and an optional "completed" or "X left" text. The progress bar color changes to onSurface when completed, primary otherwise.

EpisodeDetailItem

Large detail card for episode browsing. Column layout at 200dp width with a 200dp square image (extraLarge corners), subtitle line (date and duration/feed), title (1 line), and HTML description (4 lines). Spacing of 4dp between elements with an extra 8dp after the image.

EpisodeClipItem

Full-viewport clip item for TikTok-style browsing. Fills the entire available space with a blurred background image (20dp blur radius), a semi-transparent overlay (0.6 alpha background color), and centered content: a 250dp elevated card with the episode image, title (3 lines), HTML description (6 lines), and a bottom row with clip animation text, like toggle, and play/pause toggle.
