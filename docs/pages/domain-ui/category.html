CategoryButton

Interactive category selector used in the onboarding flow. 140dp square Surface with extraLarge shape (28dp corners). Features an animated background image with blur effect when selected, a vertical gradient overlay (transparent to 0.8 surface alpha), animated text size (16sp to 22sp), animated Y offset (45dp to 0dp), and a toggle icon (add/check) in the top-right corner. When selected, a 2dp primary border is applied.

CategoryItem

Simple category card for browsing views. Column layout with a 140dp square image (largeIncreased shape, 16dp corners) and a text label below with 8dp spacing. Uses bodyMedium typography, 1-line max with ellipsis.
