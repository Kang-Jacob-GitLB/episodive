ChannelItem

Channel card with dominant color extraction. Column layout at 250dp width with extraLarge shape (28dp corners). Contains a square image (1:1 aspect ratio) on top and an 80dp description box below. The description box background uses the dominant color extracted from the image at 0.5 alpha, with DominantRegion.Bottom specified for color extraction. Text is centered, titleSmall typography, 3-line max.

Implementation Details

The dominant color extraction is performed by the StateImage component via the onDominantColorExtracted callback with DominantRegion.Bottom specified. This extracts the dominant color from the bottom portion of the image, which provides a natural visual transition between the image and the description box. The extracted color is stored in a local mutableStateOf and applied at 0.5 alpha as the box background. Before extraction completes, Color.DarkGray is used as the default.
