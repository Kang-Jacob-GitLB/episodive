EpisodiveIconToggleButton

A toggleable icon button that switches between unchecked (transparent) and checked (filled primary) states. Used for like/follow interactions.

Showcase

Unchecked
Checked

Properties

ParamTypeDefault
checkedBooleanrequired
onCheckedChange(Boolean) -> Unitrequired
modifierModifierModifier
enabledBooleantrue
icon@Composable () -> Unitrequired
checkedIcon@Composable () -> Unitrequired

Compose Example

EpisodiveIconToggleButton(
    checked = isLiked,
    onCheckedChange = { onToggleLike(episodeId) },
    icon = {
        Icon(TablerIcons.Heart, contentDescription = "Like")
    },
    checkedIcon = {
        Icon(TablerIcons.HeartFilled, contentDescription = "Unlike")
    }
)

Compose Example

EpisodiveIconProgressButton(
    onClick = { onPlayPause() },
    isLoading = isBuffering,
    icon = {
        Icon(
            imageVector = if (isPlaying) TablerIcons.Pause else TablerIcons.Play,
            contentDescription = "Play/Pause"
        )
    }
)
