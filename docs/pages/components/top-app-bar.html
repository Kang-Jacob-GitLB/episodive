EpisodiveTopAppBar

Standard top app bar with left-aligned title. Supports a navigation icon on the left and optional action icons on the right.

Showcase

Podcast Detail
Navigation + Title
Now Playing
Navigation + Title + Action Icon

Properties

ParamTypeDefault

modifierModifierModifier

colorsTopAppBarColorsM3 defaults

Compose Example

EpisodiveTopAppBar(
    title = { Text("Podcast Detail") },
    navigationIcon = {
        EpisodiveIconButton(
            onClick = onNavigateBack,
            icon = {
                Icon(TablerIcons.ArrowLeft, contentDescription = "Back")
            }
        )
    },
    actions = {
        EpisodiveIconButton(
            onClick = onShowMenu,
            icon = {
                Icon(TablerIcons.DotsVertical, contentDescription = "More")
            }
        )
    }
)

EpisodiveCenterTopAppBar

Center-aligned top app bar variant. Title is centered between the navigation and action areas. Used for screens with a focused, symmetrical header.
