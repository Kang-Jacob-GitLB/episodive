Buttons

Filled, Outlined, and Text button variants

core/designsystem/src/main/kotlin/.../component/Button.kt

EpisodiveButton

Primary filled button used for the most prominent actions. Uses primary background with onPrimary text color and full rounded corners.

Showcase

Default
With Icon
Disabled

Compose Example

EpisodiveButton(
    onClick = { /* handle click */ },
    modifier = Modifier,
    enabled = true,
    text = { Text("Subscribe") },
    leadingIcon = null
)

EpisodiveOutlinedButton

Outlined button for secondary actions. Transparent background with a 1dp border in outline color.

Showcase

Default
Disabled

Compose Example

EpisodiveOutlinedButton(
    onClick = { /* handle click */ },
    modifier = Modifier,
    enabled = true,
    text = { Text("Following") }
)

EpisodiveTextButton

Text-only button for tertiary or inline actions. No background, minimal horizontal padding.

Showcase

Default
Disabled

Compose Example

EpisodiveTextButton(
    onClick = { /* handle click */ },
    modifier = Modifier,
    enabled = true,
    text = { Text("See All") }
)
