EpisodiveScaffold

Top-level layout container combining a TopAppBar with a Scaffold. Provides consistent screen structure with a title, optional navigation and action icons, and a scrollable content area.

Compose Example

EpisodiveScaffold(
    title = "Podcast Details",
    navigationIcon = TablerIcons.ArrowLeft,
    actionIcon = TablerIcons.DotsVertical,
    onNavigationClick = { navController.popBackStack() },
    onActionClick = { showMenu = true },
    scrollBehavior = scrollBehavior
) { paddingValues ->
    LazyColumn(contentPadding = paddingValues) {
        // content

SectionHeader

A horizontal header row for content sections. Displays a title with optional trailing action icon. Uses headlineSmall typography by default with 16dp padding.

FadeTopBarLayout

A layout that fades in a top bar as the user scrolls down. The top bar starts fully transparent and becomes opaque after scrolling past a configurable offset (default 700). Supports both LazyListState and LazyGridState scroll tracking.

Showcase

Podcast Title
scroll position: 0 (top)
Top bar alpha: 0.0
At top (alpha = 0)
Podcast Title
scroll position: ~350
Top bar alpha: 0.5
Mid-scroll (alpha = 0.5)
Podcast Title
scroll position: 700+
Top bar alpha: 1.0
Past offset (alpha = 1.0)

Properties

ParamTypeDefault
offsetInt700
listStateLazyListStatevariant 1
gridStateLazyGridStatevariant 2
topBar@Composable (alpha: Float) -> Unitrequired
content@Composable () -> Unitrequired

Compose Example

val listState = rememberLazyListState()

FadeTopBarLayout(
    offset = 700,
    listState = listState,
    topBar = { alpha ->
        EpisodiveScaffold(
            title = "Podcast Title",
            modifier = Modifier.alpha(alpha)
        )
    }
) {
    LazyColumn(state = listState) {
        // scrollable content
    }
}

FadeTopBarLayout

offset: 700 (default) alpha: 0.0 .. 1.0 scroll source: LazyListState scroll source: LazyGridState calculation: min(offset, scrollOffset) / offset
