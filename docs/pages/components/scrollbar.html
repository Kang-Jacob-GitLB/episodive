Scrollbar

Draggable and decorative scrollbar components with state management

DraggableScrollbar

An interactive scrollbar that users can drag to navigate content quickly. Supports both vertical and horizontal orientations. The thumb has a minimum size of 40dp (default) and responds to drag gestures. Long-press triggers continuous scrolling with a 10ms delay and 2% delta per press event.

Showcase

Compose Example

val listState = rememberLazyListState()
val scrollbarState = rememberScrollbarState(listState)

Box {
    LazyColumn(state = listState) {
        items(100) { index ->
            ListItem(text = "Item $index")
        }
    }

    DraggableScrollbar(
        state = scrollbarState,
        orientation = Orientation.Vertical,
        onThumbMoved = { percent ->
            coroutineScope.launch {
                listState.scrollToItem((percent * 100).toInt())
            }
        },
        modifier = Modifier.align(Alignment.CenterEnd)
    )
}

DecorativeScrollbar

A non-interactive, thin scrollbar indicator for visual feedback only. 2dp wide (vertical) or 2dp tall (horizontal). Does not respond to touch input -- purely decorative to show scroll position.

Showcase

Thin decorative indicator
Draggable (4dp)
Decorative (2dp)
Width comparison

Compose Example

val listState = rememberLazyListState()
val scrollbarState = rememberScrollbarState(listState)

Box {
    LazyColumn(state = listState) {
        items(50) { index ->
            ListItem(text = "Item $index")
        }
    }

    DecorativeScrollbar(
        state = scrollbarState,
        orientation = Orientation.Vertical,
        modifier = Modifier.align(Alignment.CenterEnd)
    )
}

ScrollbarState

Holds the state for scrollbar positioning. Tracks the thumb size relative to the track, the thumb position, and the visible track percentage. Used by both DraggableScrollbar and DecorativeScrollbar.

Properties

ParamTypeDescription
thumbSizePercentFloatSize of the thumb as a fraction of the track (0.0 to 1.0). Smaller values indicate more content.
thumbMovedPercentFloatPosition of the thumb along the track (0.0 = start, 1.0 = end).
thumbTrackSizePercentFloatPercentage of the total content that is currently visible in the viewport.

Compose Example

val scrollbarState = ScrollbarState(
    thumbSizePercent = 0.3f,
    thumbMovedPercent = 0.5f,
    thumbTrackSizePercent = 0.3f
)

// Or use the remember helper:
val listState = rememberLazyListState()
val scrollbarState = rememberScrollbarState(listState)
