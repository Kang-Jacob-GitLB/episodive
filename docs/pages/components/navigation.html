Compose Example

EpisodiveNavigationBar {
    destinations.forEach { dest ->
        EpisodiveNavigationBarItem(
            selected = currentRoute == dest.route,
            onClick = { onNavigate(dest.route) },
            icon = { Icon(dest.icon, contentDescription = dest.label) },
            selectedIcon = { Icon(dest.selectedIcon, contentDescription = dest.label) },
            label = { Text(dest.label) }
        )
    }
}

EpisodiveTab & EpisodiveTabRow

core/designsystem/src/main/kotlin/.../component/Tab.kt

Horizontal tab layout with an active indicator. EpisodiveTabRow provides the container with a bottom border, while EpisodiveTab renders individual tabs.

Compose Example

EpisodiveTabRow(selectedTabIndex = selectedTab) {
    tabs.forEachIndexed { index, title ->
        EpisodiveTab(
            selected = selectedTab == index,
            onClick = { selectedTab = index },
            text = { Text(title) }
        )
    }
}
