EpisodiveViewToggleButton

A toggle button that switches between expanded and collapsed views. Displays different text labels and an icon circle that rotates to indicate the current state. Content padding is 16dp start, 8dp top/bottom, 12dp end. The trailing icon sits inside a 38dp circle with surfaceContainerHigh background.

Compose Example

var expanded by remember { mutableStateOf(false) }

EpisodiveViewToggleButton(
    expanded = expanded,
    onExpandedChange = { expanded = it },
    enabled = true,
    text = { Text("Episodes") },
    compactText = { Text("Show More") },
    expandedText = { Text("Show Less") }
)

EpisodiveDragHandle

A small horizontal bar indicator used at the top of bottom sheets to signal drag-to-dismiss affordance. Centered horizontally with 16dp vertical padding.
