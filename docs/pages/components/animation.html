WaveAnimationIcon

An animated audio visualizer made of vertical bars that oscillate at different offsets. Default configuration uses 5 bars, each 2dp wide with 2dp spacing. Maximum bar height is 16dp. Animation uses infinite tween (300-500ms) with reverse repeat mode, each bar offset for a staggered effect.

Showcase

Animating (isAnimating = true)
Static (isAnimating = false)
Custom color (primary)

Properties

ParamTypeDefault
barCountInt5
colorColorMaterialTheme.colorScheme.onSurface

Compose Example

WaveAnimationIcon(
    barCount = 5,
    color = MaterialTheme.colorScheme.onSurface,
    isAnimating = isPlaying
)

EpisodiveIconText

A compact inline layout pairing an icon with text, separated by a 6dp gap. Commonly used for metadata display like episode counts, durations, and categories.

EpisodiveIconText(
    icon = TablerIcons.Clock,
    text = "45 min"
)

EpisodiveIconText(
    text = "12 episodes",
    trailingIcon = TablerIcons.ArrowRight
)

ClipAnimationIconText

A pill-shaped indicator combining the wave animation with text. Used as a "Now Playing" overlay on clip/soundbite cards. Has a CircleShape container with semi-transparent surface background (0.3 alpha), 8dp vertical and 16dp horizontal padding, and 8dp gap between animation and text.

Compose Example

    isAnimating = isPlaying,
    text = if (isPlaying) "Now Playing" else "Paused"
)
