Episodive는 Tabler Icons 라이브러리를 사용합니다. 모든 아이콘은 EpisodiveIcons object에서 중앙 관리되며, 각 아이콘은 Tabler 아이콘에 매핑됩니다. 아이콘은 24dp 크기를 기본으로 사용하며 stroke width 2를 적용합니다.

Usage Guidelines

Size

모든 아이콘은 기본 24 x 24dp 크기를 사용합니다. IconButton 내부에서는 40 x 40dp 터치 영역이 보장됩니다.

Color

tint 파라미터로 색상을 지정합니다. 기본값은 LocalContentColor를 따르며, Primary 또는 OnSurface 색상을 주로 사용합니다.

Filled vs Outline

Bottom Navigation에서는 선택 상태에 Filled 아이콘을, 비선택 상태에 Outline 아이콘을 사용합니다.

Accessibility

모든 아이콘에는 contentDescription을 지정하여 스크린리더 접근성을 보장합니다.

Code Example

// Icon usage
Icon(
    imageVector = EpisodiveIcons.Play,
    contentDescription = "Play",
    tint = MaterialTheme.colorScheme.primary
)

// Inside IconButton
IconButton(onClick = { /* ... */ }) {
    Icon(
        imageVector = EpisodiveIcons.Like,
        contentDescription = "Like"
    )
}

// Bottom Navigation selected/unselected
val icon = if (selected) {
    EpisodiveIcons.HomeFilled
} else {
    EpisodiveIcons.Home
}
