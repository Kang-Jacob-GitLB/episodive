Elevation in Dark Theme

Material3의 Dark Theme에서는 그림자 대신 surface tint overlay를 사용하여 고도를 표현합니다. 높은 고도의 surface일수록 Primary 색상의 tint가 더 많이 적용되어 밝아집니다. 이는 어두운 배경에서 그림자가 잘 보이지 않기 때문입니다.

Code Example

// Spacing usage
Spacer(modifier = Modifier.height(EpisodiveDimension.lg)) // 16dp

Column(
    verticalArrangement = Arrangement.spacedBy(EpisodiveDimension.sm) // 8dp
) { /* ... */ }

Row(
    modifier = Modifier.padding(horizontal = EpisodiveDimension.lg) // 16dp
) { /* ... */ }

// Shape usage
Card(
    shape = MaterialTheme.shapes.medium // 12dp corner
) { /* ... */ }

// Elevation usage
Surface(
    tonalElevation = 3.dp // Level 2
) { /* ... */ }
