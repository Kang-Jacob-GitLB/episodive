Player

Full-screen audio player with dual-mode (bar + bottom sheet)

feature/player/src/main/kotlin/.../PlayerScreen.kt

Screen Overview

The Player is the most complex screen in the app, operating in two modes: a minimized PlayerBar (70dp tall, persistent at the bottom) and an expanded PlayerBottomSheet (full-screen modal). The expanded view features a gradient background derived from album art, transport controls, chapter navigation, a speed dial, and a playlist sheet.

ViewModel Notes

Uses @Player(EpisodivePlayers.Main) qualified PlayerRepository. State is composed from multiple reactive flows: getNowPlayingUseCase, playerRepository.progress, playerRepository.isPlaying, getPlaylistUseCase, getUserDataUseCase (for speed), and getChaptersUseCase. Supports background playback via MediaNotificationService.

Navigation

Any Screen → PlayerBar (persistent) → PlayerBottomSheet (tap bar) → PodcastRoute (tap podcast)

The PlayerBar is a persistent overlay rendered in the main app scaffold, above the bottom navigation bar. It is visible on all screens when playback is active. Tapping the bar expands it into a full-screen ModalBottomSheet. The player uses the @Player(EpisodivePlayers.Main) ExoPlayer instance for full episode playback.
