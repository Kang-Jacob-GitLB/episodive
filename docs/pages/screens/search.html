Screen Structure

The search screen uses an EpisodiveSearchBar that expands on tap. The collapsed state shows trending podcasts and recent episodes. The expanded state shows recent search history and live search results with a 500ms debounce.

Search States

Collapsed (Default)

The search bar is compact. Below it, two browse sections are visible:

  • Trending Podcasts — horizontal PodcastItem list from GetTrendingPodcastsUseCase(max=10)
  • Recent Episodes — vertical EpisodeItem list from GetRecentEpisodesUseCase(max=6)

Expanded (Search Active)

When the search bar is tapped, it expands to full width. The content switches to:

  • Recent Searches — list of past queries with history icon and delete button per item, plus a "Clear All" option
  • Search Results — live results as PodcastSimpleItem rows, driven by SearchUseCase with 500ms debounce

MVI Architecture

SearchState

  • Loading
  • Success(searchQuery, searchHistory, searchResult, recentSearches, categories, recentEpisodes, trendingPodcasts)
  • Error(message: String)

SearchAction

  • QueryChanged(query: String)
  • ClickSearch(query: String)
  • ClearQuery
  • ClickRecentSearch(query: String)
  • RemoveRecentSearch(query: String)
  • ClearRecentSearches
  • ClickCategory(category: Category)
  • ClickPodcast(podcast: Podcast)
  • ClickEpisode(episode: Episode)
  • ToggleLikedEpisode(episode: Episode)

SearchEffect

  • NavigateToCategory(category: Category)
  • NavigateToPodcast(podcastId: Long)
  • NavigateToEpisode(episode: Episode)

State Data Types

FieldTypeDescription

searchHistoryList<String>Reserved for future search history

recentSearchesList<String>Recently searched queries (max 100)
categoriesList<Category>All category entries for quick browse
recentEpisodesList<Episode>Recent episodes for collapsed view

The Search screen is one of the 4 bottom navigation destinations. Clicking a podcast navigates to Podcast Detail. Clicking a category navigates to Channel Detail. Clicking an episode plays it directly via PlayEpisodeUseCase without navigation.
