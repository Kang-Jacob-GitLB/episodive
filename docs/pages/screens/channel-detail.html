Screen Overview

The Channel Detail screen displays a channel (category) with its associated podcasts in a two-column grid. It uses FadeTopBarLayout with a parallax-style header image that fades and scales on scroll (offset=300). The channel header overlays the image with a gradient fade, and the footer contains a channel description and website link. The screen requires a channelId parameter via AssistedViewModel.

Parallax Scroll Effect

The channel banner image uses a custom scroll-driven parallax effect. As the user scrolls down:

PropertyFormulaDescription
alpha1f - (scrollOffset / 600f).coerceIn(0f, 1f)Image fades out as user scrolls
scale1f + (1f - scrollProgress) * 0.1fSubtle zoom-out effect (max 10%)

MVI Architecture

State

  • Loading
  • Success(channel: Channel, podcasts: List<Podcast>)
  • Error(message: String)

Action

  • ClickBack
  • ClickPodcast(podcastId: Long)

Effect

  • NavigateBack
  • NavigateToPodcast(podcastId: Long)

ViewModel

Uses AssistedInject with @Assisted("id") val id: Long. State is composed by combining getChannelByIdUseCase(id) and getPodcastsByChannelUseCase(channel) flows. Podcasts are fetched reactively via flatMapLatest once the channel data is available.

Compose Example

@Composable
internal fun ChannelRoute(
    viewModel: ChannelViewModel = hiltViewModel(),
    onBackClick: () -> Unit,
    onPodcastClick: (Long) -> Unit,
    onShowSnackbar: suspend (String, String?) -> Boolean,
) {
    val state by viewModel.state.collectAsStateWithLifecycle()

    LaunchedEffect(Unit) {
        viewModel.effect.collectLatest { effect ->
            when (effect) {
                is ChannelEffect.NavigateBack -> onBackClick()
                is ChannelEffect.NavigateToPodcast -> onPodcastClick(effect.podcastId)
            }
        }
    }

    when (val s = state) {
        is ChannelState.Loading -> LoadingScreen()
        is ChannelState.Success -> ChannelScreen(
            channel = s.channel,
            podcasts = s.podcasts,
            onBackClick = { viewModel.sendAction(ChannelAction.ClickBack) },
            onPodcastClick = { viewModel.sendAction(ChannelAction.ClickPodcast(it)) },
        )
        is ChannelState.Error -> ErrorScreen(message = s.message)
    }
}

Navigation

Home / Search → ChannelRoute(id: Long) → PodcastRoute (tap podcast)
ChannelRoute → NavigateBack (back button / ClickBack) 
// Navigation route definition
@Serializable
data class ChannelRoute(val id: Long)

// Navigate to channel detail
fun NavController.navigateToChannel(
    channelId: Long,
    navOptions: NavOptionsBuilder.() -> Unit = {},
) = navigate(route = ChannelRoute(channelId), navOptions)

// Register in NavGraphBuilder
fun NavGraphBuilder.channelScreen(
    onBackClick: () -> Unit,
    onPodcastClick: (Long) -> Unit,
    onShowSnackbar: suspend (String, String?) -> Boolean,
) {
    composable<ChannelRoute> { entry ->
        val id = entry.toRoute<ChannelRoute>().id
        ChannelRoute(
            viewModel = hiltViewModel<ChannelViewModel, ChannelViewModel.Factory>(
                key = "channel:$id"
            ) { factory -> factory.create(id) },
            onBackClick = onBackClick,
            onPodcastClick = onPodcastClick,
            onShowSnackbar = onShowSnackbar
        )
    }
}
